An App Playground

This site is run by Steve Perkins, an enterprise Java developer and author of the book Hibernate Search by Example.

In my personal time, I like to tinker with polyglot technologies that I might not get a chance to use extensively at work. Things like Go, Clojure, and Node.js on the server-side. Advanced JavaScript on the client side. Mobile apps, etc.

So basically, JiffyStudios.com is the hipster alter-ego for StevePerkins.net. It hosts those learning examples that are polished enough to show publicly. Click left and right through this carousel to review the publicly available sites and apps.

Domain Jiffy

A tool for coming up with domain name ideas. Seperates itself from other generators by being extremely customizable, letting you mix and match "elements" such as word synonyms, common words, etc.

This site gave me a chance to work with Node.js and Express, drag-and-drop with jQuery UI, and some of the same language processing techniques that I researched while writing my Hibernate Search book.
